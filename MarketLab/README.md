# MarketLab – Financial Market Simulator

MarketLab is a web-based quantitative trading simulator that models stock price 
movements using stochastic processes and allows users to trade virtual assets 
in a risk-free environment.

---

## 🚀 Version History
- **v5**: Dashboard UI + Portfolio analytics (Return, Sharpe, Drawdown)  
- **v4**: GBM update + Volatility display  
- **v3**: Graph toggle  
- **v2**: Sell button + Net Worth  
- **v1**: AAPL chart + Buy button  

---

## ✨ Features
- Real-time stock price simulation  
- Geometric Brownian Motion (GBM) modeling  
- Virtual trading (buy/sell)  
- Portfolio tracking (cash, shares, net worth)  
- Volatility calculation per stock  
- Historical price tracking  
- Random market shocks  

### 📊 Analytics
- Portfolio return (%)  
- Sharpe ratio  
- Maximum drawdown  
- Portfolio performance chart  

### 🎨 UI
- Dark-mode dashboard  
- Slide-in portfolio panel  
- Quick summary bar (cash, shares, net worth)  

---

## 🛠 Technologies
- HTML, CSS, JavaScript  
- Chart.js  

---

## 🧠 Model
- μ: drift  
- σ: volatility  
- Δt: time step  
- Z: standard normal variable  

---

## ▶️ Run
Open `index.html` in browser.
